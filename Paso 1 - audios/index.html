Prueba de audio para alertas de somnolencia

Esta página sirve para validar que una futura web de detección de somnolencia pueda emitir alertas desde el audio del celular, incluyendo salida por audífonos Bluetooth o por el sistema de audio del auto.

Recomendación: Asegúrate de que el volumen del celular y del dispositivo Bluetooth esté suficientemente alto.

Los navegadores móviles normalmente requieren interacción del usuario para reproducir audio. Por eso, toca un botón para iniciar la prueba.

Esta web no puede leer de forma confiable el volumen físico real del teléfono o del Bluetooth, así que la validación del volumen se hace de manera guiada según lo que tú escuches.
